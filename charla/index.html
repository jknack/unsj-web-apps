MWA: Arquitectura de una aplicación web moderna

  • Es un conjunto de buenas prácticas para construir aplicaciones web en Java
  • Basada en: Servlet 3.0, Spring 3.2.x with Maven 3.x
  • Minimo tiempo de creación y configuración
  • Basada en modulos para incrementar productividad

Tecnología utilizada en MWA

  • Java Servlet 3.x
  • Inyección de dependencies
  • REST JSON API con Spring MVC 3.x
  • JPA 2.x / Mongo / Neo4j / QueryDSL / etc
  • Maven 3.x

Fundamentos de MWA

  • Evitar XML para inyección de dependencias
  • Ambiente/Entorno de aplicación
  • Reutilizar Software mediante el uso de modulos

HTML 5

  • Conjunto de especificaciones y standares para el desarrollo de aplicaciones Web
  • Tecnologias obsoletas: Flash (Flex), Active.X, Applets
  • Notables mejoras en el engine de visualización: en especial WebKit

Por que JavaScript?

  • Casi tan antiguo como la web misma y uno de los mejores lenguajes jamas creados
  • Notables mejoras en los motores/engine de JavaScript: en especial v8 (Google Chrome)
  • jQuery everywhere
  • Extensa biblioteca con miles y miles de librerias
  • Aplicación de MVC en el navegador
